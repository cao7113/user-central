#==Welcome to user-central project!

This project aims some targets after forking from https://github.com/joshsoftware/sso-devise-omniauth-provider:
* user authentication
* profile management for end-users and back-end platform
* sso solution and user service for other (inner-)applications
* role based access control(future)
